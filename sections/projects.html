Research Projects

TrumbleMOF

Transformer MLIP for Long-Range Interactions in MOFs

TrumbleMOF is a Transformer-based machine learning interatomic potential developed for metal-organic frameworks. The model is designed to capture long-range interactions and complex atomic environments in porous materials, enabling accurate and scalable atomistic simulations for large MOF systems.

Technologies: Transformer Architectures, Machine Learning Interatomic Potentials (MLIPs), Atomistic Simulation, Materials AI

Status: Active

Reduced Modelling for Unstructured Mesh

AI-Accelerated Scientific Simulation

Developing reduced-order models for simulations on unstructured meshes using dimensionality reduction and data-driven surrogate modelling. The approach aims to approximate high-fidelity numerical simulations with significantly lower computational cost, enabling faster analysis of complex physical systems.

Technologies: Reduced-Order Modelling, Machine Learning, Scientific Computing, High-Performance Computing

Status: Active

ML Insight for H₂O Capture Materials Design

ML Insight for H₂O Capture Materials Design

Utilizing machine learning insights to design novel H₂O capture materials, integrating predictive modeling with materials synthesis for enhanced adsorption performance and accelerated innovation in water capture technologies.

Technologies: Machine Learning Insights, Explainable AI, Adsorption Modeling, Computational Chemistry

Status: Active

BuildMOF

Multimodal Targeted De Novo MOF Generation

Developing advanced generative models for multimodal, targeted, and de novo generation of metal-organic frameworks (MOFs). The project integrates multiple data modalities (text, structure, properties) with targeted generation techniques to create novel MOFs with specific desired characteristics, enabling precise and efficient materials discovery.

Technologies: Multimodal AI, Targeted Generative Models, De Novo Design, Computational Chemistry

Status: Planning

ReadMOF

A machine learning model that "reads" MOFs using chemical text (rather than graph structures)

The key to ReadMOF lies in the MOF to text construction approach, transforming MOF problems from geometric modeling to language understanding. It naturally aligns with chemical language + graph/geometry interactions.

Technologies: Cheminformatics, Materials Language Modeling, NLP Transformer, Efficient Vector Regression

Status: Completed

P-STMAE

Physics-Informed Spatiotemporal Masked Autoencoder

Developing a physics-aware masked autoencoder for efficient spatiotemporal system prediction, combining transformer architectures with physical constraints for improved accuracy and interpretability.

Technologies: Transformers, Masked Autoencoders (MAE), Physics-Informed Neural Networks (PINNs), Spatiotemporal Modeling

Status: Completed

Generative Models for Microfluidics

Deep Learning for Drop Coalescence Prediction

The key challenge of this project is handling imbalanced datasets. Developed generative models and explainable AI frameworks to predict and analyze drop coalescence phenomena in microfluidic devices, addressing data imbalance issues to enable better device design and optimization.

Technologies: Variational Autoencoders (VAE), Explainable AI, Computational Fluid Dynamics, Imbalanced Learning

Status: Completed
